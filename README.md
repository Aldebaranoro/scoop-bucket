> # Scoop Bucket
>
> A Bucket of all my tools.

## Installation

```shell
> scoop bucket add Aldebaranoro https://github.com/Aldebaranoro/scoop-bucket
```

> scoop bucket add Aldebaranoro https://github.com/Aldebaranoro/scoop-bucket
> scoop install team-divider
